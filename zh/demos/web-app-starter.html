小型网页应用入门

Prototype April 28, 2026

概述

一个用于快速构建和测试基于 Web 的工具的最小化、生产就绪模板。无需框架，无需构建步骤——只有随处可运行的干净 HTML、CSS 和 JavaScript。

问题

当你有一个 Web 工具的想法时，现代开发通常需要：

  • 搭建 React/Vue/Angular 项目
  • 配置 webpack/vite/parcel
  • 安装数十个 npm 包
  • 学习框架特定的模式

等你能开始编写真正想法的代码时，已经在搭建上花费了数小时。

解决方案

一个极简的起步模板，提供：

  • 干净、语义化的 HTML 结构
  • 使用现代特性的响应式 CSS
  • 无依赖的模块化 JavaScript
  • 轻松部署（只需上传文件）

在几分钟内从想法到可用的原型，而非几小时。

项目结构

web-app-starter/
├── index.html          # 主应用页面
├── styles.css          # 所有样式（无需预处理器）
├── app.js              # 应用逻辑
├── utils.js            # 辅助函数
└── README.md           # 使用说明

就这些。五个文件。零依赖。

JavaScript 架构

工具模块（utils.js）

const Utils = {
  getElement(id) {
    const el = document.getElementById(id);
    if (!el) console.error(`未找到 ID 为 "${id}" 的元素`);
    return el;
  },
  
  showLoading(element) {
    element.innerHTML = '<div class="loading">处理中...</div>';
  },
  
  showError(element, message) {
    element.innerHTML = `<div class="error">❌ ${message}</div>`;
  },
  
  formatDate(date) {
    return new Date(date).toLocaleDateString('zh-CN', {
      year: 'numeric', month: 'long', day: 'numeric'
    });
  },
  
  debounce(func, wait) {
    let timeout;
    return function executedFunction(...args) {
      const later = () => { clearTimeout(timeout); func(...args); };
      clearTimeout(timeout);
      timeout = setTimeout(later, wait);
    };
  }
};

主应用（app.js）

class WebApp {
  constructor() {
    this.initializeElements();
    this.bindEvents();
    this.loadData();
  }
  
  initializeElements() {
    this.form = Utils.getElement('input-form');
    this.input = Utils.getElement('user-input');
    this.output = Utils.getElement('output-area');
  }
  
  bindEvents() {
    if (this.form) {
      this.form.addEventListener('submit', (e) => {
        e.preventDefault();
        this.processInput();
      });
    }
  }
  
  async processInput() {
    const value = this.input.value.trim();
    if (!value) {
      Utils.showError(this.output, '请输入一些内容');
      return;
    }
    Utils.showLoading(this.output);
    try {
      const result = await this.performCalculation(value);
      this.displayResult(result);
    } catch (error) {
      Utils.showError(this.output, error.message);
    }
  }
  
  async performCalculation(input) {
    return new Promise((resolve) => {
      setTimeout(() => {
        resolve({
          original: input,
          processed: input.toUpperCase(),
          length: input.length,
          timestamp: new Date().toISOString()
        });
      }, 500);
    });
  }
  
  displayResult(data) {
    this.output.innerHTML = `
      <div class="result-card">
        <h3>处理完成</h3>
        <div class="result-details">
          <p><strong>原始:</strong> ${data.original}</p>
          <p><strong>处理后:</strong> ${data.processed}</p>
          <p><strong>长度:</strong> ${data.length} 字符</p>
          <p><strong>时间:</strong> ${Utils.formatDate(data.timestamp)}</p>
        </div>
      </div>
    `;
  }
}

document.addEventListener('DOMContentLoaded', () => {
  window.app = new WebApp();
});

部署选项

方案1: GitHub Pages（免费）

git init && git add . && git commit -m "初始提交"
git remote add origin https://github.com/username/repo.git
git push -u origin main
# 在仓库设置中启用 GitHub Pages

方案2: Netlify Drop（免费）

直接将项目文件夹拖放到 netlify.com/drop。

方案3: 传统托管

通过 FTP/SFTP 上传文件到任意 Web 服务器。
