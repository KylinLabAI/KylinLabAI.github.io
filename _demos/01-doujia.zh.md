# 🫛 豆荚 — 让你真正拥有 AI 对话

**一款桌面应用，把你和 AI 的每一次对话变成可搜索、可整理、可导出的知识资产。**

</div>

---

## 你遇到过这些问题吗？

你在豆包、通义千问上积累了几百条有价值的对话，但这些知识**被锁在平台里** ——

- 找不到上周那条关键回答
- 没办法导出成文档
- 无法批量整理或清理
- 担心哪天账号出问题，对话全丢

## 豆荚的解决方案

**豆荚**把 AI 平台直接嵌入桌面应用，你像平常一样对话，豆荚在后台自动采集、存储、整理，让你对自己的知识资产拥有完全的掌控权。

---

## 核心价值

| 你得到什么 | 怎么实现的 |
|---|---|
| **对话永不丢失** | 自动采集，每一条对话实时保存到本地 |
| **即时找到任何内容** | 全文检索，跨所有对话搜索关键词 |
| **大规模整理** | 标签、AI 自动分类、主题聚类 |
| **自由导出** | Markdown、HTML、PDF，你的数据你做主 |
| **提炼知识** | AI 摘要 + 知识卡片，碎片对话变体系 |
| **离线可用** | 全部数据存在本地 SQLite，不依赖云端 |

---

## 核心功能

### 1. 内嵌 AI 对话 + 自动采集

在豆荚里直接使用豆包（或通义千问），体验和浏览器完全一致。每一条对话自动采集保存到本地，无需手动复制。

### 2. 全面的对话管理

在一个界面浏览、搜索、筛选和管理所有对话。支持按时间范围、平台、状态、标签多维度筛选。

### 3. 智能组织

- **AI 自动分类** — 一键按主题给对话打标签
- **主题聚类** — 把相关对话自动归组
- **知识图谱** — 可视化查看主题之间的关联

### 4. AI 知识提炼

- **AI 摘要** — 对任何对话生成精炼摘要
- **知识卡片** — 从一条或多条对话中提取结构化笔记
- **智能清理** — AI 推荐哪些对话可以归档或删除

### 5. 导出与分享

- 导出为 **Markdown**、**HTML** 或 **PDF**
- 设置默认导出目录，一键导出不再弹框
- **局域网分享** — 同一网络内的设备可以直接查看你的知识库

### 6. 中英双语界面

完整支持中文和英文界面，在设置中一键切换。

---

## 快速上手

1. **下载**适合你系统的安装包
2. **安装并启动**豆荚
3. **开始对话** — AI 平台在应用内加载，正常使用
4. **点击采集**（或开启自动采集）保存对话
5. 在**知识管理**标签页浏览、搜索和整理

---

## 技术栈

| 层级 | 技术 |
|------|-----|
| 桌面框架 | Electron 33 |
| 前端 | Vue 3 + Element Plus |
| 数据库 | SQLite（better-sqlite3） |
| AI 引擎 | 豆包 API / 通义千问 API |
| 构建 | electron-vite + electron-builder |

---

## 隐私与安全

- **数据全部存在本地** — 对话内容永远不会离开你的电脑
- **无遥测** — 应用不会向外发送任何数据
- **你的密钥，你的控制** — AI 智能功能使用你自己的 API Key
